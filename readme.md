# toys

Various linux utilities in a single exe file for windows

## Usage
```cmd
toys [toys options] CMD [cmd options]
```

## Toys:
 - ***acpi***: Show battery status.
 - ***base64***: base64 encode/decode data and print to standard output
 - ***basename***: Return non-directory portion of a pathname removing suffix.
 - ***cal***: Print a calendar.
 - ***cat***: Concatenate FILE(s) to standard output.
 - ***colors***: Print colors.
 - ***fd***: Find files.
 - ***file***: Determine file type.
 - ***head***: Output the first 10 lines of a file/s.
 - ***http***: Make an HTTP request.
 - ***less***: Page contents of a file.
 - ***ls***: List directory contents.
 - ***neo***: Move files or directories to trash bin.
 - ***pwgen***: Generate human-readable random passwords.
 - ***rm***: Move files or directories to trash bin.
 - ***serve***: Serve contents of directory as static web pages.
 - ***sleep***: Delay for a specified amount of time.
 - ***stat***: Display file/s status.
 - ***strings***: Display printable strings in a binary file.
 - ***tac***: Output lines in reverse order.
 - ***tail***: Output the last 10 lines of a file/s.
 - ***tee***: Copy standard input to each FILE, and also to standard output.
 - ***time***: Time a simple command.
 - ***touch***: Create a file/s.
 - ***unlock***: Unlock files or folders from other programs.
 - ***uptime***: Tell how long the system has been running.
 - ***wc***: Print newline, word, and byte counts for each file .
 - ***xargs***: Build and execute command lines from standard input.
 - ***xxd***: Dump or edit files in hexidecimal.
